# Ícones da Extensão

## O QUE FAZER AQUI:
Criar ícones para a extensão em 3 tamanhos:
- **icon16.png** — 16x16 pixels (ícone pequeno na barra do browser)
- **icon48.png** — 48x48 pixels (ícone na página de extensões)
- **icon128.png** — 128x128 pixels (ícone na Chrome Web Store)

## Como Criar:
1. Usar qualquer ferramenta de design (Figma, Canva, Paint, etc.)
2. Criar um ícone simples — ex: um sino 🔔 ou megafone 📢
3. Fundo transparente (PNG)
4. Exportar nos 3 tamanhos

## Alternativa Temporária:
Para testar sem criar ícones, podes usar ícones genéricos.
O Chrome mostrará um ícone padrão se não encontrar os ficheiros.
